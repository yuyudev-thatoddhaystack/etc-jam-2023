A freeshell based on "The Friendly Computer Bug" lapel pin. Made for the Ukagaka Dream Team 2023 Etc Jam. Comes with four dress-ups.

Font used for menu sidebar is "Computer says no"

Menu sidebar will need adjustments for different sizes, a psd is included to make increasing or decreasing the size easier.
